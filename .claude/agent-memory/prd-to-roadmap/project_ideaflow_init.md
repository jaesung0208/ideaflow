IdeaFlow 프로젝트 ROADMAP.md를 2026-03-13에 최초 생성함.

- 전체 4 Phase / 8주 계획 (각 Phase 2주 스프린트)
- Phase 1: 정적 캔버스 UI (로컬 상태) - 아직 코드 미생성 상태
- Phase 2: Firebase 연동 + 실시간 동기화 (MVP 범위)
- Phase 3: 모바일 최적화 + STT
- Phase 4: AI 클러스터링 + Vercel 배포

**Why:** 사용자가 PRD 기반으로 실행 가능한 스프린트 계획이 필요했음. 프론트엔드 우선 개발 원칙 적용.

**How to apply:** 스프린트 시작 시 해당 Phase의 작업 목록과 완료 기준을 참조. 기존 ROADMAP이 Phase 1 완료로 잘못 표시되어 있었으나 실제 코드가 없어 예정 상태로 수정함.
